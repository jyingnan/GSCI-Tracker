Index Construction

GSCI aggregates CAMEO-coded conflict events from GDELT, retaining only events with strictly negative Goldstein scores. Each event is weighted by its absolute severity and normalized by total daily news sources.

GSCIt = Σi∈Ct |Gi| × Si,t / Nt

Ct = conflict events on day t
|Gi| = absolute Goldstein weight
Si,t = sources reporting event i
Nt = total daily sources

Data Sources & Update

  • Data Source: GDELT Event Stream (100+ languages)
  • Event Taxonomy: CAMEO conflict categories (Goldstein < 0)
  • Update Schedule: Every Monday via GitHub Actions
  • Coverage: March 2015 – present
  • Complement: GPR Index (Caldara & Iacoviello 2022)
