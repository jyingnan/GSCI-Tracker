Global Severity-weighted Conflict Index (GSCI)

Real-time event-based monitoring of global conflict intensity via GDELT

Download Full CSV Data View on GitHub

Methodology

The GSCI is an objective, event-based indicator that complements perception-based geopolitical risk measures. By analyzing millions of records from the GDELT Project, the index identifies physical conflict events (Goldstein Scale < 0) and weights them by their global media footprint and inherent severity.

  • Data Source: GDELT Event Stream (Multilingual)
  • Update Frequency: Every Monday (Automated via GitHub Actions)
  • Coverage: March 2015 – Present
